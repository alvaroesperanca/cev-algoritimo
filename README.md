Curso de algorítimo do Curso em Vídeo

Tipos:

* Inteiro
* Real / Numerico
* Literal / Caractere
* Logico

Operadores:

\+  : Adição  
\-  : Subtração  
\*  : Multiplicação  
/ : Divisão  
\ : Divisão Inteira  
^ : Exponenciação  
% : Módulo  


Ordem de precedência

* () : Parênteses
* ^ : Exponenciação
* \* / : Multiplicação / Divisão
* +- : Adição / Subtração

Funções aritméticas

* Abs: Valor absoluto
* Exp: Exponenciação
* Int: Valor inteiro
* RaizQ: Raiz quadrada
* Pi: Retorna Pi
* Sen: Seno (rad)
* Cos: Cosseno (rad)
* Tan: Tangente (rad)
* GraupRad: Graus para radiano
